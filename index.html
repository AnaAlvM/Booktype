Welcome to the Booktype dev pages.

* Join dev list

To collaborate on the code contact us over irc or email and let us know what you are doing and what are your thoughts on it. We can talk about the best way to go forward and help you get to know the code base. Then check wiki for info, future plans and current branches.

To submit a contribution fork the code on github, make changes and send us pull request. If you don't use github (for whatever reason), fork it locally and send us your patch over email or give us access to your repository.
