To collaborate on the code, please contact us on GitHub or by email and let us know what you are doing. We can talk about the best way to go forward and help you get to know the code base.

To submit a contribution, please fork the code on GitHub, make changes and send us a pull request. If you don't use GitHub, please fork it locally and send us your patch by email, or give us access to your repository.
